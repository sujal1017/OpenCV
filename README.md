# 🖥 OpenCV Learning Project (Tutorial-Based)

This project was built by following an OpenCV tutorial video and implementing all the concepts step by step in Python.

The objective of this repository is to strengthen my understanding of OpenCV fundamentals and practice real-time image and video processing.

---

## 📚 Topics Covered

- Reading and displaying images
- Resizing and cropping images
- Drawing shapes and adding text
- Converting color spaces (BGR to Grayscale)
- Image thresholding
- Edge detection (Canny)
- Blurring and smoothing
- Video capture using webcam
- Saving processed images/videos

---

## 🛠 Tech Stack

- Python 3.x
- OpenCV (cv2)
- NumPy

---

## ▶ How To Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/OpenCV-Tutorial-Project.git
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run any Python file:
   ```bash
   python filename.py
   ```
